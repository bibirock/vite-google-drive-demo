## 此個人專案作品採用技術如下

-   前端框架： Vite + Vue3 + Pinia
-   串接第三方 API: Google Drive API
-   主要使用語言： TypeScript + Javascript
-   CSS 預處理器： Tailwind CSS + SASS
-   UI library： ant-design-vue
-   HTML 模板: PUG
-   GitHub 部署：gh-pages
-   多語系系統： i18n
-   plugins : mitt、axios、iconify
